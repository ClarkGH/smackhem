## 13. Data Formats

### 13.1 Map Data (JSON)

```json
{
  "id": "overworld_01",
  "chunks": [
    {
      "id": "chunk_0_0",
      "bounds": [0,0,0, 16,16,16],
      "meshes": [
        { "type": "cube", "pos": [2,0,2], "scale": [1,1,1] }
      ]
    }
  ]
}
```

Pure data. No logic. Portable forever.
